# vimfiles

```bash
    git clone https://github.com/emarkham/vimfiles.git ~/.vim  
    ln -sf ~/.vim/vimrc ~/.vimrc  
    ln -sf ~/.vim/gvimrc ~/.gvimrc  
    cd ~/.vim  
    git submodule init  
    git submodule update  
```
